# 🕵️ 時空特務：攔截怪盜X (SpyGame-X)

這是一個專為國小四年級學生設計的 **「時間與分秒運算」** 互動學習遊戲。透過與虛擬角色「怪盜X」的對話與心理攻防，將枯燥的數學練習轉化為緊張刺激的特務任務。

## 🌟 特色
- **沉浸式情境**：怪盜X 會根據學生的回答進行即時挑釁或嘲諷。
- **針對性設計**：題庫涵蓋「時間加法」、「經過時間」與「分秒減法」，特別強化孩子最容易出錯的「進位」與「借位」邏輯。
- **零依賴開發**：純 HTML/JavaScript 編寫，不需安裝任何環境，點開即玩。
- **戰績結算**：遊戲結束後提供完整報告，方便老師或家長檢視孩子的學習盲點。

## 🎮 如何執行
1. 下載本專案的 `index.html`。
2. 使用任何現代瀏覽器（Chrome, Edge, Safari）開啟該檔案。
3. 開始攔截怪盜X！

## 🎯 教學應用
- **課堂暖身**：作為數學課前 5 分鐘的趣味練習。
- **遠距教學**：透過螢幕分享，由全班共同討論答案。
- **自主學習**：學生可在家中反覆練習，挑戰全對紀錄。

## 🛠️ 技術細節
- **前端語法**：Vanilla JavaScript, CSS3 (Flexbox, CSS Variables), HTML5。
- **設計模式**：單頁面應用 (SPA) 邏輯。
